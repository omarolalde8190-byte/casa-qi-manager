GUIA SIMPLE PARA SUBIR A GITHUB

1. Descomprime este ZIP.
2. Entra a la carpeta descomprimida.
3. Selecciona TODO lo que está adentro.
4. En GitHub entra al repositorio casa-qi-manager.
5. Add file > Upload files.
6. Arrastra TODOS los archivos y carpetas.
7. Asegúrate de que NO aparezca ningún archivo llamado download.
8. Presiona Commit changes.

IMPORTANTE:
Debes subir también las carpetas:
- icons
- fotos
