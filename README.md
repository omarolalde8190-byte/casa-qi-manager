# Casa Qi Manager

Sistema interno de operación para Casa Qi Hotel Boutique.

## Incluye

- Aplicación web instalable como PWA.
- Conexión a Google Sheets mediante Apps Script.
- Configuración lista para Netlify.
- Estructura limpia para GitHub.

## Archivos que deben estar en GitHub

- index.html
- config.js
- manifest.json
- service-worker.js
- netlify.toml
- package.json
- README.md
- .gitignore
- carpeta icons
- carpeta fotos
